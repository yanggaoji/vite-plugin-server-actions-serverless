Server Functions Made Simple

Write server-side functions and call them from your client code as if they were local. Built-in validation, TypeScript support, and automatic API documentation.

Get Started View Examples 
import { z } from 'zod';

// Define validation schema
const TodoSchema = z.object({
  text: z.string().min(1).max(100),
  priority: z.enum(['low', 'medium', 'high']).optional(),
});

// Server function with automatic validation
export async function addTodo(todo) {
  const newTodo = {
    id: Date.now(),
    ...todo,
    completed: false,
    createdAt: new Date()
  };
  
  await db.todos.create(newTodo);
  return newTodo;
}

// Attach schema for validation
addTodo.schema = z.tuple([TodoSchema]);

Type-Safe by Default

Full TypeScript support with automatic type inference between server and client code.

Built-in Validation

Automatic request validation using Zod schemas. Define once, validate everywhere.

Auto API Docs

Generates OpenAPI spec and Swagger UI automatically from your server functions.

Framework Agnostic

Works with any frontend framework - React, Vue, Svelte, or vanilla JavaScript.

Zero Config

Works out of the box with sensible defaults. Progressive enhancement when you need it.

Production Ready

Optimized builds with tree-shaking, code splitting, and Express server generation.

Get Started in Minutes

1 Install the plugin

npm install vite-plugin-server-actions

2 Configure Vite

// vite.config.js
import { serverActions } from 'vite-plugin-server-actions';

export default {
  plugins: [serverActions()]
}

3 Create server functions

// actions.server.js
export async function greet(name) {
  return `Hello, ${name}!`;
}
